Numbers

Use numbers directly in your source code. Do not use quotation marks as they are for strings. Integer is a number that cannot contain decimal. But float can.
integer = 42
float = 4.2
a = b = c = d = 0.0
Integer cannot start with 0... except it is 0. But float can.
x = 90
y = 090.00

Operators

Operators are special symbols in Python that carry out arithmetic or logical computation. The value that the operator operates on is called the operand.
Arithmetic Operators
Operator Meaning Example

- Subtract right operand from the left or unary minus x-y-2
* Multiply two oprands x*y
/ Divide left operand by the right one(always results into float) raise
