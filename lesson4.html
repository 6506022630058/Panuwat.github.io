Len, Upper, Lower, Format

Len

Example:

fruit = 'apple'
fruit_len = len(fruit)
print(fruit_len)

Output:

5

Upper

Example:

fruit = 'apple'
print(fruit.upper())

Output:

APPLE

Lower

Example:

fruit = 'Apple'
print(fruit.lower())

Output:

apple

Format

Example:

first = 'I'
second = 'love'
print('{} {} {}'.format(first,second,'Python'))

Output:

I love Python

String Methods

Join

Example:

text = ('I love python')
print('/'.join(text))
print(''.join(reversed(text)))

Output:

I/ /l/o/v/e/ /p/y/t/h/o/n
nohtyp evol I

Split

Example:

text = ('I love python')
print(text.split())
print(text.split('o'))

Output:

['I', 'love', 'python']
['I l', 've pyth', 'n']

Strip

Example:

text = ('#1#I love python###')
print(text.strip('#'))
print(text.lstrip('#'))
print(text.rstrip('#'))

Output:

1#I love python
1#I love python###
#1#I love python

Find

Example:

text = ('I love python')
print(text.find('ve'))

Output:

4

Replace

Example:

text = ('I like python')
print(text.replace('like','love'))

Output:

I love python

Endswith

Example:

files = ['test1.py','test2.txt','test3.doc']
for file in files:
if file.endswith('.txt'):
print(file)
if file.endswith('.doc'):
print(file)
if file.endswith('.py'):
print(file)

Output:

test1.py
test2.txt
test3.doc

Boolean Test

Method True if
.isalnum() String consists of only alphanumeric characters (no symbols)
.isalpha() String consists of only alphabetic characters (no symbols)
.isnumeric() String consists of only numeric characters
.istitle() String is in title case
.isupper() String's alphabetic characters are all upper case
.islower() String's alphabetic characters are all lower case
.isspace() String consists of only whitespace characters

Example:

alnum1 = 'aB1'
alnum2 = 'aB1/'
print(alnum1.isalnum()) ---> True
print(alnum2.isalnum()) ---> False
alpha1 = 'aB'
alpha2 = 'aB1'
print(alpha1.isalpha()) ---> True
print(alpha2.isalpha()) ---> False
numeric1 = '12'
numeric2 = '12.3'
numeric3 = '12a'
print(numeric1.isnumeric()) ---> True
print(numeric2.isnumeric()) ---> False
print(numeric3.isnumeric()) ---> False
title1 = 'I Love Python'
title2 = 'I love Python'
print(title1.istitle()) ---> True
print(title2.istitle()) ---> False
upper1 = 'ABC123/'
upper2 = 'AbC123'
print(upper1.isupper()) ---> True
print(upper2.isupper()) ---> False
lower1 = 'abc123/'
lower2 = 'aBc123'
print(lower1.islower()) ---> True
print(lower2.islower()) ---> False
space1 = ' '
space2 = ''
space3 = 'a'
print(space1.isspace()) ---> True
print(space2.isspace()) ---> False
print(space3.isspace()) ---> False
