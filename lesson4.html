Len, Upper, Lower, Format

Len

Example:

fruit = 'apple'
fruit_len = len(fruit)
print(fruit_len)

The output will be:

5

Upper

Example:

fruit = 'apple'
print(fruit.upper())

The output will be:

APPLE

Lower

Example:

fruit = 'Apple'
print(fruit.lower())

The output will be:

apple

Format

Example:

first = 'I'
second = 'love'
print('{} {} {}'.format(first,second,'Python'))

The output will be:

I love Python
