Introduction to Functions

Introduction to Functions

- A function is a section of reusable code that performs an action.
- A function has a name and is called, or executed, by that name.
- Optionally, functions can accept arguments and return data.

Function naming rules

- Cannot use key words as a function name
- Cannot contain spaces
- First character must be a letter or underscore
- All other characters must be a letter, number or underscore
- Uppercase and lowercase characters are distinct

Advantages of Function

- Reducing duplication of code
- Decomposing complex problems into simpler pieces
- Improving clarity of the code
- Reuse of code
- Information hiding

Example:

def message():
print('I Love Python')
print('Python Love Me')
message()

Output:

I Love Python
Python Love Me

Passing Arguments

Local variables

Example:

def main():
num2()
num1()
def num1():
number = 10
print('num1 =',number)
def num2():
number = 5
print('num2 =',number)

Output:

num2 = 5
num1 = 10

Passing arguments

Example:

def main():
num = 10
double(num)
def double(number):
result = number * 2
print(result)
main()

Output:

20

Example:

def main():
Multiply(3, 5)

Global Variables and Global Constants

Example:

number = 0
def main():
global number
number = 5
def show_number():
print(number)
show_number()
main()
show_number()

Output:

0
5

Value-Returning

Example:

def main():
item_price = get_price()
item_quantity = get_quantity()
print('Total is',get_total(item_price, item_quantity))
def get_price():
price = float(input('Enter a price: '))
return price
def get_quantity():
quantity = int(input('How many: '))
return quantity
def get_total(price, quantity):
total = price * quantity
return total
main()

Output:

Enter a price: 5.25
How many: 4
Total is 21.0
